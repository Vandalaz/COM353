Task 2

I added an unordered list with 3 items. Each of these was the name of a colour.

By adding mouseover events to each of these list items, I could change the colour based on the users selection.

I added a mouseleave event on the unordered list element itself so that the main content div would return to white after the user was finished playing about.

2 of these events were created with the colour name and 1 was created using the rgb function.

With Colour Name:

onmouseout="document.getElementById('content').style.backgroundColor = 'White';"

With RGB:

onmouseover="document.getElementById('content').style.backgroundColor = 'Gray';"
