Task 1 - Analyze and Explain

var myAge =45;
Initialises a variable called myAge as an integerwith the value 45.

myAge += 4;
Addition assignment that adds right operand to the left. myAge's value is now 49.

myAge ++;
Increment (Postfix) operator that simply increase myAge by 1. It is now 50.

myAge == 40+10;
This is an comparison operator that checks if myAge's value is equal to 50 (40 + 10). While this would be true, the result is not assigned anywhere.

myAge <=38;
Another comparison operator, this time the code is checking if myAge is less than or equal to 38. Is it not so the result would be false.

myAge =40;
Here, we assign the value of 40 to myAge.
