Explanation of validNo()

  1. If statement checks if value is present in the input.
  2. If false, show an alert, focus on the input, return false.
  3. If there is a value present, begin executing the else clause.
  4. Initialise a few empty strings and also a variable that holds the value in the phone input.
  5. Begin a for loop which will loop over the user input.
  6. 1 character at a time, check if the character is a number between 0 and 9. If it is, add it to the numbersOnly string.
  7. Do this until we have looped over all the input values.
  8. If numbersOnly is not 13 digits, show an alert, focus on the field, and return false.
  9. If the number is indeed 13 digits, begin performing substrings to pull values out into variables.
  10. After setting variables for area code, leading0, exchange, ext1, and ext2; concatenate them into a new format.
  11. Populate the input field with this new string and return true.

Explanation of validEmail()

  1. Check if email is set in the form.
  2. If not, show an alert, focus on the email field, and return false.
  3. If the email value is present, execute the else branch.
  4. Initalise emailAddress variable to the value input by the user.
  5. Use indexOf to find the index of the characters @ & . in the input string.
  6. An if statement checks that these index values are greater than 0 and thus the characters are present.
  7. It also checks to make sure that they are not the only 2 values input.
  8. If these conditions have been satisfied, return true.
  9. Otherwise, show an alert, focus on the email field, and return false.
